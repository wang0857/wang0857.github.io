Learning Background

I’m an international student originally from Taiwan[1] and graduated with a bachelor’s degree in economics from National Taipei University[2]. After graduation, I worked as a Marketing Officer for four years. When working in the field of Marketing, I communicated with many designers for different types of marketing materials, e.g., promotion websites, online application form, DM, eDM and content videos. That was my first time to know about interactive media and website design.

For web design, I did encounter some confusing problems when I was testing the website with web designers. For example, our company set older version browser as a default, yet the web designer designed a website that could only be seen in new version browser. As a result, no matter how many times I refreshed the tested website, it never showed a right layout. After many communications and testing with the web designer, eventually we figured out the solution which was to adjust my browser’s setting. Within this experience, I found it a sense of accomplishment to figure out the solution of a website. Especially when I looked the code in Chrome Developer Tools with the web developer, I thought those codes were cool, so I hoped that I could learn this language!


Start to Learn HTML, CSS, and JavaScript, and Come to Canada

After many considerations, I resigned from my company, and started to learn those basic website languages such as HTML, CSS and JavaScript. During self-learning, I was also searching for a related program to study with others because I thought that it would inspire me with more ideas to design a beautiful website. The second reason for me to study a second program was to have a relative learning background of website design. While I was searching for programs which were related to web design, I found Interactive Media Design program of Algonquin College. The course list of IMD program was amazing because it contained many types of media design. What’s more, there were many resources in the college to help students find a job after graduation. I thought it was exactly what I wanted, so I applied for this program and came to Canada for studying this program ultimately.
